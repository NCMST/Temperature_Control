\doxysubsection{Detailed Description}
A class to handle temperature readings from a K type thermocouple and an NTC thermistor. 

This class provides methods to read temperatures from a K type thermocouple using the Gyver\+MAX6675 sensor and from an NTC thermistor. It abstracts the hardware-\/specific details and provides a simple interface for obtaining temperature readings.

\begin{DoxyNote}{Note}
Ensure that the CLK\+\_\+\+PIN, DATA\+\_\+\+PIN, and CS\+\_\+\+PIN are correctly defined for the Gyver\+MAX6675 sensor.
\end{DoxyNote}
\begin{DoxyAttention}{Attention}
This class assumes that the necessary hardware connections are made and the Gyver\+MAX6675 library is included.
\end{DoxyAttention}
\begin{DoxyWarning}{Warning}
The temperature readings are dependent on the accuracy of the sensors and the correct configuration of the pins.
\end{DoxyWarning}
