\doxysubsection{Detailed Description}
A class to implement a \mbox{\hyperlink{classPID}{PID}} controller. 

The \mbox{\hyperlink{classPID}{PID}} class provides methods to set \mbox{\hyperlink{classPID}{PID}} coefficients, compute the output value, and set limits for the output value. It also provides getter and setter methods for the \mbox{\hyperlink{classPID}{PID}} coefficients.

\begin{DoxyNote}{Note}
The compute method should be called periodically to update the output value. 
\end{DoxyNote}
