\doxysubsection{Detailed Description}
A class to manage the OLED screen using Adafruit\+\_\+\+SSD1306 library. 

This class provides methods to initialize, clear, and print text on the screen.

\begin{DoxyNote}{Note}
Inherits from Adafruit\+\_\+\+SSD1306.
