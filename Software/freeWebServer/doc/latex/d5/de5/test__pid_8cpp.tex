Tests the \mbox{\hyperlink{classPID}{PID}} controller with output limits. 

This function creates a \mbox{\hyperlink{classPID}{PID}} controller with specific gains and sets output limits. It then tests the \mbox{\hyperlink{classPID}{PID}} controller with input values that exceed the set limits to ensure the output is correctly limited.

The \mbox{\hyperlink{classPID}{PID}} controller is initialized with the following gains\+:
\begin{DoxyItemize}
\item Proportional gain (Kp)\+: 1.\+0
\item Integral gain (Ki)\+: 0.\+1
\item Derivative gain (Kd)\+: 0.\+05
\end{DoxyItemize}

The output limits are set as follows\+:
\begin{DoxyItemize}
\item Minimum limit\+: 0.\+0
\item Maximum limit\+: 10.\+0
\end{DoxyItemize}

The function performs the following tests\+:
\begin{DoxyItemize}
\item Computes the \mbox{\hyperlink{classPID}{PID}} output for an input value of 20.\+0 with a setpoint of 5.\+0. The expected output should be limited to 10.\+0.
\item Computes the \mbox{\hyperlink{classPID}{PID}} output for an input value of -\/20.\+0 with a setpoint of 5.\+0. The expected output should be limited to 0.\+0.
\end{DoxyItemize}

The tests use the TEST\+\_\+\+ASSERT\+\_\+\+FLOAT\+\_\+\+WITHIN macro to check if the computed output is within 0.\+01 of the expected limited value.
